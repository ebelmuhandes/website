# موقعي الشخصي

موقع ويب شخصي احترافي مع لوحة تحكم، مبني بـ PHP و SQLite.

## المتطلبات
- PHP 7.4+
- خادم ويب (Apache/Nginx)
- دعم SQLite

## التثبيت
1. استخرج الملفات في مجلد على الخادم
2. تأكد من صلاحيات الكتابة على مجلد `data/` و `uploads/`
3. افتح الموقع في المتصفح

## بيانات الدخول الافتراضية
- اسم المستخدم: `admin`
- كلمة المرور: `admin123`
- **قم بتغيير كلمة المرور فوراً!**

## المميزات
- تصميم عصري داكن مع تأثيرات زجاجية
- خلفية متحركة وتأثيرات بصرية
- تصميم متجاوب بالكامل
- لوحة تحكم كاملة
- إدارة الأقسام والمشاريع والرسائل
